# Retail Sales Data Analysis

This project analyzes a retail sales dataset to uncover business insights about customer purchasing behavior, product performance, and revenue trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## Key Insights
- Monthly revenue fluctuates around a stable average.
- Customers aged 46–60 generate the most revenue.
- A small number of customers contribute significantly to total sales.

## Visualizations
Charts included in this analysis:
- Monthly Sales Trend
- Revenue by Gender
- Revenue by Age Group
- Top Spending Customers
- Retail Sales Dashboard

## Project Structure
data/ - dataset used for analysis
notebooks/ - Python analysis notebook
visuals/ - charts generated from the analysis
