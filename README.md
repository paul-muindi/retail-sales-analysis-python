# Retail Sales Data Analysis

This project analyzes a retail sales dataset to uncover business insights about customer purchasing behavior, product performance, and revenue trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## Key Insights
- Identified the highest revenue generating product categories
- Analyzed customer demographics and buying patterns
- Visualized sales trends over time
- Built charts to communicate insights effectively

## Project Structure
data/ - dataset used for analysis
notebooks/ - Python analysis notebook
visuals/ - charts generated from the analysis

Data Cleaning
Exploratory Data Analysis
